# 打包前检查清单

## Windows 11 行为检查

- 确认托盘图标可见，左键点击能唤起快速记录窗口
- 确认全局快捷键在应用后台运行时仍可唤起窗口
- 确认提醒时间变更后，新的调度会按新设置生效
- 确认基础日报与 AI 润色日报都能复制到系统剪贴板
- 确认 AI 配置、提醒配置和历史任务在应用重启后仍可读取

## 发布前配置检查

- `identifier`、`productName`、`version` 与实际发布版本一致
- 图标资源存在并可用于窗口与托盘
- 构建命令 `npm run build` 与 `cargo test --manifest-path src-tauri/Cargo.toml` 均通过
- 生成包前确认不输出 API Key、Base URL、模型名到日志或错误弹窗

## 发布说明建议

- 说明支持 Windows 11 桌面环境
- 说明 AI 润色依赖用户自行配置的 OpenAI 兼容接口
- 说明离线情况下仍可进行任务记录、汇总编辑和基础日报生成
